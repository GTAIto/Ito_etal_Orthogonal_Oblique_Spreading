# Ito\_etal\_Orthogonal\_Oblique\_Spreading



This repository accompanies the following manuscript:



*Intrusive Magmatism, Plate Stresses, and the Cause of Orthogonal Versus Oblique Spreading Mid-Ocean Ridges* (2026),

Garrett Ito (gito@hawaii.edu), Jana Schierjott, Mark D. Behn, Xiaochuan Tian, Boris J. P. Kaus and Anton A. Popov, Geochem. Geophys. Geosyst.



# Overview

This repository has all the input files and MATLAB scripts needed to reproduce Figures 2, 4, 5, 7, 8, 9, 10, 11, and 12. Visit each folder and read "Readme.txt", which points to the folder containing the LaMEM input file, gives instructions on how to reproduce the model solutions, and identifies the MATLAB files needed to reproduce the figure.  The videos from Figures 2 and 4 are in the respective folders.

# 

# Software

The 3D numerical models were produced using the open-source geodynamics software LaMEM (Lithosphere and Mantle Evolution Model, https://github.com/UniMainzGeo/LaMEM), version 2.2.0, provided in this repository and on Zenodo (https://zenodo.org/records/20048335). The MATLAB code used to generate the 2D elastic models is also included.
